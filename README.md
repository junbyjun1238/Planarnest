# PlanarNest

Public release repository for **PlanarNest**.

PlanarNest is a desktop nesting tool for laser/CNC workflows.
It focuses on practical DXF/SVG import, stable placement, and clean export for production use.

## Key Features

- DXF/SVG part import
- Nesting with spacing/rotation controls
- Multi-sheet workflow
- Remnant/offcut usage
- SVG/DXF export
